Overview

surface_dipole_library is a production-ready Python library for computing dipole moments and dipole–dipole interaction strengths between surface atoms of a glass substrate and a thin film.

Its primary use-case is pre-screening glass–thin-film interfaces before expensive DFT simulations. The library takes partial charges and atomic positions as input and returns a ranked table of all pairwise interactions, identifying which substrate–film atom pairs produce the strongest attraction (most negative energy) and which are repulsive.

Design principle: The glass substrate is fixed — the user only supplies the film atoms. The canonical 12-atom borosilicate glass surface with ClayFF partial charges is loaded automatically.

Scientific Background

When a thin film is deposited on a glass substrate, the initial adhesion is governed by electrostatic interactions between surface dipoles. Atoms with high partial charges near the interface create local dipole moments that interact according to classical dipole–dipole physics.

Dipole Moment

The electric dipole moment of a collection of point charges is computed as:

\[ \boldsymbol{\mu} = \sum_i q_i \, \mathbf{r}_i \quad \text{(units: } e \cdot \text{\AA)} \]

where \(q_i\) is the partial charge in units of elementary charge \(e\) and \(\mathbf{r}_i\) is the Cartesian position in Angstroms. The conversion factor is 1 e·Å ≈ 4.803 Debye.

Effective Local Atomic Dipole

Rather than using the global dipole vector directly, each atom's local contribution is computed relative to the group's geometric centroid \(\mathbf{r}_c\):

\[ \boldsymbol{\mu}_i^{\text{local}} = q_i \, (\mathbf{r}_i - \mathbf{r}_c) \]

This makes all interaction energies independent of the global coordinate origin — translating both surfaces by any vector does not change any computed energy.

Dipole–Dipole Interaction Energy

The interaction energy between two point dipoles separated by displacement vector \(\mathbf{r}\) is:

\[ U = \frac{1}{4\pi\varepsilon_0} \left[ \frac{\boldsymbol{\mu}_1 \cdot \boldsymbol{\mu}_2}{r^3} - \frac{3\,(\boldsymbol{\mu}_1 \cdot \hat{\mathbf{r}})\, (\boldsymbol{\mu}_2 \cdot \hat{\mathbf{r}})}{r^3} \right] \]

In practical surface-science units (μ in e·Å, r in Å, U in eV):

\[ U = K_e \left[ \frac{\boldsymbol{\mu}_1 \cdot \boldsymbol{\mu}_2}{r^3} - \frac{3\,(\boldsymbol{\mu}_1 \cdot \mathbf{r})\,(\boldsymbol{\mu}_2 \cdot \mathbf{r})}{r^5} \right] \]

where the Coulomb constant \(K_e = 14.3996 \text{ eV·Å/}e^2\) is derived from: 

k_e = 8.9876e9 N·m²/C²
e   = 1.6022e-19 C
1 Å = 1e-10 m
1 eV = 1.6022e-19 J
K_e = k_e × e² / Å = 14.3996 eV
Sign Convention
U < 0
Attractive — energetically favourable
U ≈ 0
Neutral — negligible interaction
U > 0
Repulsive — energetically unfavourable

Glass Substrate

The substrate is fixed as a canonical 12-atom borosilicate glass surface derived from the β-cristobalite SiO2 (001) surface unit cell (lattice parameter a = 7.16 Å). Partial charges are from the ClayFF force field (Cygan et al. 2004), cross-validated against CHARMM-silica (Lopes et al. 2006) and INTERFACE FF (Heinz et al. 2013).

Input Format

Film atoms are provided as a Python list of dict objects. Each dict has three required keys:

KeyTypeUnitsDescription
element str — Chemical symbol, e.g. "Ti", "O", "In"
charge float elementary e Partial charge — positive for cations, negative for anions
position list[3] Å Cartesian coordinates [x, y, z]
label str — Optional. Custom atom label. Auto-generated if omitted (e.g. film_Ti1)
Coordinate convention: All atoms must be in the same Cartesian frame. The glass substrate occupies z ≈ 0–2.35 Å. Film atoms should start at z > 1.6 Å, typically z ≈ 3.5 Å.
Example
film_atoms = [
    {"element": "Ti", "charge":  0.580, "position": [0.00, 0.00, 3.50], "label": "Ti5c1_film"},
    {"element": "Ti", "charge":  0.580, "position": [2.96, 0.00, 3.50], "label": "Ti5c2_film"},
    {"element": "O",  "charge": -0.290, "position": [1.48, 0.00, 4.20], "label": "O_br1_film"},
    {"element": "O",  "charge": -0.260, "position": [0.00, 1.62, 3.80], "label": "O_ip1_film"},
]

API Reference

analyze_film_on_glass(film_atoms, cutoff=None, glass_variant="borosilicate", verbose=True)

Run the full dipole–dipole interaction analysis with the substrate fixed as borosilicate glass. The user only supplies the thin film.

Parameters
ParameterTypeDefaultDescription
film_atoms list[dict] required Film surface atoms. Each dict: element, charge, position, label (optional).
cutoff float or None None Maximum atom–atom distance (Å) to include. None = all pairs.
glass_variant str "borosilicate" Glass composition to use. See Glass Variants.
verbose bool True Print formatted summary table to stdout.
Returns

dict — see Output section for full key list.

analyze_surface_interactions(substrate_atoms, film_atoms, cutoff=None, verbose=True)

General-purpose entry point where the user provides both substrate and film.

analyze_from_xyz(substrate_file, film_file, cutoff=None, has_charges=True, verbose=True)

Load both surfaces from 5-column XYZ files (element x y z charge) and run analysis.

analyze_from_ase(substrate_file, film_file, substrate_charges=None, film_charges=None, cutoff=None, verbose=True)

Load both surfaces via ASE (supports .cif, POSCAR, .xyz, .vasp, etc.) and run analysis.

Quick start example
from surface_dipole_library import analyze_film_on_glass

tio2_film = [
    {"element": "Ti", "charge":  0.580, "position": [0.00, 0.00, 3.50]},
    {"element": "Ti", "charge":  0.580, "position": [2.96, 0.00, 3.50]},
    {"element": "O",  "charge": -0.290, "position": [1.48, 0.00, 4.20]},
    {"element": "O",  "charge": -0.260, "position": [0.00, 1.62, 3.80]},
]

result = analyze_film_on_glass(tio2_film, cutoff=8.0)
print(result["interaction_table"])
print(result["best_interaction"])

Glass Variants

Four glass compositions are available. Pass the variant name as the glass_variant parameter:

Variant key Composition Atoms Notes
"borosilicate" SiO2 + B2O3 + silanol 12 Default. Models display/solar/optical glass (Schott, Corning)
"fused_silica" Pure SiO2 10 No boron, no silanol. Optical / UV-grade fused silica
"aluminosilicate" SiO2 + Al2O3 12 Al substitutes one Si. Models alkali aluminosilicate glass
"soda_lime" SiO2 + Na network modifier 12 Na+ as network modifier. Models common float glass
Usage
result = analyze_film_on_glass(film_atoms, glass_variant="fused_silica")
result = analyze_film_on_glass(film_atoms, glass_variant="aluminosilicate")
result = analyze_film_on_glass(film_atoms, glass_variant="soda_lime")
